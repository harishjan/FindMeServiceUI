To run the project the system will need the Node version 16 or above:

To setup project :
run>> npm install
run >> npm start
The project should start localhost:8080 or 8084 or if the port is already used it should start with another port and should print the new prot number

Note: browse using localhost and not the ip address, the google apis being used for testing has permission to access through localhost, The IP addrss will not work.
